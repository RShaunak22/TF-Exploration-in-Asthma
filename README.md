# TF-Exploration-in-Asthma
This is a repository for the files used in my MSc Project entitled: Identifying Key Transcription Factors in the Airways of Asthmatic Individuals
The directory structure of this repository is as follows:

Metadata - Holds metadata files for epithelial cell type data (from UCSC genome browser)

Outputs - Holds figures from scripts

Originals - Holds original data files for epithelial cell type and transcription factor datasets

Readmes - Citations and additional information associated with project files

Scripts - Holds the executable code alongside comments to achieve outputs
